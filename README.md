# HUGO BLOG
###### Install Dependencies
```bash
# Arch/Manjaro
sudo pacman -S hugo

# Ubuntu/Debian
sudo apt install hugo
```
see the [hugo docs](https://gohugo.io/installation/windows/) for windows installation.

###### Run Local Server
```bash
hugo run -D
```
